# Grapheme-to-Morpheme-Mapping-for-Kashmiri-written-in-the-Sharda-script
This project develops a rule-based Grapheme-to-Phoneme (G2P) conversion system specifically tailored for the Kashmiri language. A unique addition is a bidirectional transliteration module that seamlessly converts the historical Sharada script into English (Latin) characters and vice versa.
